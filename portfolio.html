Dreamify

A C#/.NET console app consiting of a client and an own built API that interracts with the Spotify API as an external API along with database connections and testing

Allowing the user to create a user profile and connect songs, genres and artists to their user. Either manually through text-input or via search querys that uses the Spotify API to get results from their database and allow the user to select songs and artist to connect to their user.

Features
-User authentication
-Account management (view, create, remove)
-Money transactions (withdraw, deposit, transfer, currency convertion)
-Menu navigation with keyboard arrows

"The Bank"

A bank Console-app with Database connections made in C#/.NET.

This C# console application simulates basic bank/atm functionalities. Users can log in to their accounts, view account details, create new accounts, withdraw, deposit, and transfer money between accounts. Accounts supports six different currencies with automatic conversion between accounts when transfering money. Additionally, there is an admin menu for managing users in the app.

Features
-User authentication
-Account management (view, create, remove)
-Money transactions (withdraw, deposit, transfer, currency convertion)
-Menu navigation with keyboard arrows

Console Warrior

A little side project I started in the beginning of my programming journey to work on incorporating the knowledge I gained so far since I started studying Fullstack.NET development in september 2023.

At the point of creation it was to practice OOP and classes, testing out inheritance in different forms. Due to excitment the practice turned into a working little game! It is by no means complete but fully playable. With combat, based on the oldschool "dice roll" where everything is chance based. Everything from the enemies you encounter, if you're gonna hit and how much xp is given.

The story is you take on the role as one of three adventurers. To venture around in a fantasy world fighting various creatures and exploring the mysterious and magical world.

Console Blackjack

A work in progress game made for fun and to practice working with APIs.

Rules are simple, you get cards dealt and chooses when to stop, when to get more cards, how much to bet etc. Just like in real life. Just without the money part.
